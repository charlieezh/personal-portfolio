Project Experience

This program leverages Python, HTML, and CSS to process HTML templates, JSON data, and other miscellaneous inputs to generate static websites. Further, I have developed server-side interactive pages that interface with a SQL database, supporting dynamic page content, CRUD operations, and session management. Additionally, I have created a client-side application in JavaScript capable of operating within a web browser. This application performs AJAX calls to communicate with a REST API, enabling a seamless user experience without page reloads.

I calculated a Minimum Spanning Tree to identify the subset of edges that ensures all locations in the zoo are connected at the lowest possible weight cost. Additionally, I implemented the Branch and Bound Algorithm, a method used to calculate the weight of an optimal solution to the Travelling Salesperson Problem, ensuring the most efficient route is found for traversing through the different locations within the zoo.

I designed and built a C++ program that simulates Euchre, a popular card game in Michigan. The program's structure is organized using Abstract Data Types (ADTs) across different files: Pact, Card, Player, and the driver file Euchre, to manage the complexities of the game logic. Leveraging the principles of Object-Oriented Programming, I used Derived Classes, Inheritance, and Polymorphism to create two distinct player types: an AI player that operates autonomously and a human-controlled player, each with unique playing strategies. These implementations facilitate a dynamic gaming experience that closely mimics the real-life interactions of Euchre players.
